# Конфигурационное управление – Вариант №7

## Описание
Инструмент визуализации графа зависимостей для менеджера пакетов Alpine Linux (APK).

## Требования
- Python 3.8+
- Установленные зависимости из `requirements.txt`

## Установка
```bash
git clone https://github.com/ВАШ_ЛОГИН/config-management-v7.git
cd config-management-v7
pip install -r requirements.txt
